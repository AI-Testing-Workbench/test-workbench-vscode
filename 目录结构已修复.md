## 问题

复制 AionUI 时目录结构不正确：

```
❌ 错误的结构：
out/aionui/
  ├── main/
  ├── preload/
  └── renderer/

✅ 正确的结构：
out/aionui/
  └── dist/
      ├── main/
      ├── preload/
      └── renderer/
```

## 已修复

```bash
# 删除错误的目录
rm -rf out/aionui

# 创建正确的结构
mkdir -p out/aionui/dist
cp -r extensions/aionui-main/out/* out/aionui/dist/
```
