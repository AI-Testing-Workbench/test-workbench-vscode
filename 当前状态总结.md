## ⚠️ 需要用户验证的问题

### IPC 数据返回格式

**现象**：
- IPC 调用返回 `true`（布尔值）而不是预期的数据对象
- 这是因为 AionUI 的 IPC handler 返回 `emitter.emit()` 的结果

**可能的影响**：
- UI 可能显示但某些功能不工作
- 或者 UI 有自己的处理方式，功能完全正常

**需要测试**：
1. UI 是否正常显示
2. 是否可以看到 Gemini agent
3. 是否可以添加智能体
4. 是否可以创建对话和发送消息
